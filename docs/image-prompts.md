## Tips para mejores resultados

1. **Midjourney**: agregá `--ar 16:9 --v 6` al final del prompt
2. **DALL-E 3**: funciona mejor con prompts descriptivos en inglés
3. **Leonardo AI**: usá el preset "Cinematic" o "Photorealistic" para mockups más realistas
4. **Canva**: podés recrear los diseños manualmente con plantillas UI
5. **Excalidraw / Figma**: si preferís schemas dibujados a mano, usá el prompt: `hand-drawn wireframe style, whiteboard sketch of exam interface`

Los mockups van en `docs/assets/` y se referencian desde el README como `![Preview](docs/assets/mockup.png)`.
